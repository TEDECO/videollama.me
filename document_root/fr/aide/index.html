Comment préparer le mobile ou la tablette de l’ancienne ?

L’objectif est que la personne âgée puisse se connecter par appel vidéo en appuyant simplement sur 4 boutons :


L’icône du cœur pour accéder à l’application

Le bouton vert pour démarrer l’appel

Le bouton rouge pour raccrocher l’appel

Le «cercle» de la barre de statut pour recommencer

Le minimum

1º Créer une icône sur l’écran d’accueil, supprimer le reste et le laisser centré sur l’écran

Comment puis-je faire cela?

2º Ouvrez l’application pour la première fois pour permettre l’utilisation de la caméra et du microphone

Autres suggestions

Augmentation du volume au maximum

Augmenter la taille des polices (lettre)

Supprimer toutes les applications possibles

Masquer les applications qui ne peuvent pas être désinstallées de l’écran d’accueil

Désactiver les notifications pour toutes les applications

Désactiver la serrure d’écran

Désactiver le PIN dans les options de sécurité

Fixez les économies d’énergie à 10 minutes ou plus

Désactiver les mises à jour automatiques

Laissez seulement l’icône à videollama.me
